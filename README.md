# HalalAudio
## что это?
HalalAudio это утилита позволяющая сделать передозный 'ремикс' на любой трек. Он автоматически добавит случайные эффекты и другие звуки к вашему аудио что бы получить хайп ✅

## как юзать
1. Поставьте зависимости через pip: numpy pydub tqdm <br>
2. Клонируйте репозиторий, добавьте свои звуки в папку `sounds` (по умолчанию там только макан)
3. Используйте: `python HalalAudio.py [путь к треку]`
